Apa yang Dibangun

Platform loyalitas belanja barang bekas khusus Indonesia.

ThriftChain dirancang untuk komunitas fashion bekas yang ingin mendapatkan nilai ekonomi nyata dari aktivitas belanja mereka. Semua transaksi dikomputerisasi dalam jaringan Solana, sehingga token $THRIFT menjadi hadiah cepat, murah, dan dapat ditukar.

  • Token $THRIFT sebagai insentif loyalitas.
  • AI rekomendasi produk berbasis Python untuk hasil lebih relevan.
  • Dompet tersemat otomatis via Privy SDK untuk pengguna Web2.
  • Cashback SOL dan konversi ke saldo Rupiah.
